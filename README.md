# Graham Towing & Recovery LLC

Professional one-page website for Graham Towing & Recovery LLC, a full-service towing company in Baytown, TX.

## Tech Stack

- **HTML5** — Semantic markup with structured data (JSON-LD)
- **CSS3** — Custom properties, CSS Grid, Flexbox, scroll animations
- **Vanilla JavaScript** — Zero dependencies, IntersectionObserver, smooth scroll
- **Google Fonts** — Inter (body) + Oswald (headings)
- **Font Awesome 6** — Icons via CDN

## Deployment

This site is designed to be deployed on **Cloudflare Pages**:

1. Push to the `main` branch of this repository
2. Connect repository in Cloudflare Pages dashboard
3. Build command: (none — static site)
4. Output directory: `/` (root)
